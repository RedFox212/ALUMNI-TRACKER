# 🦁 LATS — Lyceum Alumni Tracking System

Welcome to the **Lyceum Alumni Tracking System (LATS)**. This is a full-stack PHP/MySQL application developed for the Lyceum of Alabang.

## 🚀 Quick Start (How to Run)

Since this is a PHP application, it requires a local server environment (like XAMPP) to run.

1.  **Install XAMPP**: If you don't have it, download and install [XAMPP](https://www.apachefriends.org/index.html).
2.  **Move the Folder**: Move this entire `ALUMNI` folder into your XAMPP's `htdocs` directory:
    *   Path: `C:\xampp\htdocs\ALUMNI`
3.  **Start Services**: Open the **XAMPP Control Panel** and click **Start** for both **Apache** and **MySQL**.
4.  **Launch**: Open your browser and go to:
    *   `http://localhost/ALUMNI`
5.  **Auto-Setup**: The system will automatically create the database and import all 20 alumni records (from `seed.sql`) on your first visit!

## 🛠 Tech Stack
*   **Backend**: Vanilla PHP (8.x) with PDO.
*   **Database**: MySQL (MariaDB).
*   **Frontend**: Tailwind CSS (CDN), Inter Fonts.
*   **Icons**: Heroicons (SVG).
